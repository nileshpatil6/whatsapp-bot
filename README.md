# WhatsApp Bot

This is a WhatsApp bot project.

## Configuration

To run this project, you need to set up several environment variables. Copy `.env.example` to `.env` and fill in the values.

### WhatsApp Access Token

For the bot to work permanently without the token expiring, you need a **Permanent Access Token**.
Follow the instructions in [WHATSAPP_TOKEN_GUIDE.md](./WHATSAPP_TOKEN_GUIDE.md) to generate one.

## Setup

1. `npm install`
2. `npm run dev`
